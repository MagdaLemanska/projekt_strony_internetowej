    • Instalowanie Live Server


    Co daje nam rozszerzenie Live Server?

    Rozszerzenie sprawia, że strona odświeża się w czasie rzeczywistym!
    Pozwoli nam to oszczędzić czas, klawisz F5, oraz umożliwi nam bardziej swobodną zabawę kodem.



    Jak zainstalować Live Server?

    1. W VS Code przechodzimy do "Extensions" (Ctrl+Shift+X). W polu wyszukiwania wpisujemy "Live Server". Klikamy "Install".




    2. Przechodzimy do "Settings" (Ctrl+,). W polu wyszukiwania wpisujemy "auto save". Zmieniamy opcje automatycznego zapisywania na "afterDelay".




    3. W prawym dolnym rogu VS Code klikamy na "Go Live" lub w edytorze kodu po naciśnięciu prawego przycisku myszy wybieramy opcje "Open with Live Server" (Alt+L następnie Alt+O).
