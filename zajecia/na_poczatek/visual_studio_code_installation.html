    Po co nam Visual Studio Code?

    Strony internetowe można pisać nawet w notatniku!
    Jednakże kodowanie w edytorach kodu jakim jest między innymi Visual Studio Code ma wiele zalet.
    Pozwala przyśpieszyć nam pracę... oraz sprawić że kodowanie jest dużo przyjemniejsze...



    Jak zainstalować Visual Studio Code?

    1. Przejdź na stronę klikając w link Visual Studio Code.

    2. Z listy rozwijanej pobierz plik dla systemu którego używasz.




    3. Zainstaluj pobrany plik.

    4. Uruchom Visual Studio Code Przejdź do "Plik" następnie "Nowy Plik".




    5. Wybierz język HTML.




    6. Rozpocznij swoją przygodę z programowaniem stron.
